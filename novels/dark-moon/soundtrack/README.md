# 月球暗面 Soundtrack

Put finished audio files in `audio/` and update `catalog.js`.

Suggested structure:

- `audio/01-lunar-black-water.mp3`
- `audio/02-correction-chamber.mp3`
- `cover.jpg` or `cover.png` if a soundtrack cover is needed later

The reading page uses `window.darkMoonSoundtrack` from `catalog.js`. Leave `src` empty while a track is only a concept.
