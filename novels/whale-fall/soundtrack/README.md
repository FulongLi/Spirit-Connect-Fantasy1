# 巨鲸陨落 Soundtrack

Put finished audio files in `audio/` and update `catalog.js`.

Suggested structure:

- `audio/01-mountain-of-dead.mp3`
- `audio/02-whale-falls.mp3`
- `cover.jpg` or `cover.png` if a soundtrack cover is needed later

The reading page uses `window.whaleFallSoundtrack` from `catalog.js`. Leave `src` empty while a track is only a concept.
